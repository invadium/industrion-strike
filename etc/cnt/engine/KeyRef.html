basic control
Right key right turn (positive yaw)
Left key left turn (negative yaw)
Up key down turn (negative pitch)
Down key up turn (positive pitch)
a increase speed
d decrease speed
s match speed with 0
x match speed with cruise speed of current unit
d match speed with current target
f adjust speed to current speed value
fire control
c fire with lasers
v fire missile or torpedo
b fire missile or torpedo with binding camera
camera control
Shift+q list camera views
Shift+1-9 set camera view
q unbind camera (make camera free)
w bind camera to previous unit
e bind camera to next unit
t switch between automatic and manual control
Enter bind camera to current target
Space (only if camera free) bind camera to unit near crosshair
Enter (only if camera free) create a navigation point
target control
Space (only if camera binded) target unit near crosshair
[ target previous unit
] target next unit
p list navigation objects (nav points, buoys, stations etc.)
o list foe objects
i list friend objects
u select nearest enemy targeting you
y select nearest enemy attacking you target
quick camera and target control
Shift + F5-F12 store current target
Ctrl + F5-F12 store camera binding
F5-F12 restore target or camera binding
task control
j reset orders
; set current target as primary target
' set current target as secondary target
k set attack order
l set follow order
1-4 set agression level 1-4
flight plan control
< set previous flight plan
> set next flight plan
radar control
+ increase scale
- reduce scale
mouse wheel zoom in/out
left click select target
right click jump inside ship
left dclick select primary target
right dclick select secondary target
game control
Esc finish the mission
F1 game help
F2 key reference
F3 mission objectives
F4 on/off statis space
Ctrl-T change timing x1, x2, x4
Ctrl-P pause
